# Modul 1 – Client Acquisition System
## 1.5 Prompt-Library: Akquise & Angebote

---

## PROMPT 01: Kaltakquise-E-Mail generieren

```
Schreibe eine Kaltakquise-E-Mail für einen Freelancer im DACH-Raum.

MEIN ANGEBOT: [z.B. Content Marketing für B2B SaaS]
ZIELKUNDE: [Firmenname, Branche, Größe]
ANSPRECHPARTNER: [Name, Position]
KONKRETER AUFHÄNGER: [z.B. "Habe gesehen, dass Ihr Blog seit 3 Monaten keinen neuen Beitrag hat"]

Regeln:
- Maximal 6 Sätze
- Erster Satz = Aufhänger (kein "Mein Name ist...")
- Ein konkreter Nutzen
- Eine klare, niedrigschwellige CTA (kein "Termin buchen")
- Siezen
- Kein Anhang, kein Link
- Betreffzeile: Kurz, neugierig machend, kein Spam-Trigger
```

## PROMPT 02: LinkedIn-Nachricht (Erstkontakt)

```
Schreibe eine LinkedIn-Kontaktanfrage-Nachricht (max. 300 Zeichen).

ZIEL: Verbindung aufbauen, NICHT verkaufen.
PERSON: [Name, Position, Firma]
GEMEINSAMER NENNER: [z.B. gleiche Branche, gemeinsamer Kontakt, gleiche Gruppe]

Die Nachricht soll:
- Persönlich wirken (kein Copy-Paste-Gefühl)
- Einen konkreten Bezug haben
- KEINE Verkaufsabsicht zeigen
- Mit einer offenen Frage enden
```

## PROMPT 03: Erstgespräch vorbereiten

```
Ich habe in 30 Minuten ein Erstgespräch mit einem potenziellen Kunden.

FIRMA: [Name]
BRANCHE: [Branche]
WEBSITE: [URL]
ANSPRECHPARTNER: [Name, Position]
ANFRAGE: [Was wurde angefragt?]

Erstelle mir:
1. 3 smarte Fragen, die ich stellen sollte (die zeigen, dass ich mich vorbereitet habe)
2. 3 mögliche Painpoints basierend auf der Branche
3. 1 konkreten Quick-Win-Vorschlag, den ich im Gespräch nennen kann
4. Red Flags, auf die ich achten sollte
```

## PROMPT 04: Preisverhandlung vorbereiten

```
Ein Kunde möchte über den Preis verhandeln.

MEIN ANGEBOT: [Preis] € netto
KUNDE SAGT: "[Einwand des Kunden einfügen]"
MEIN MINIMUM: [Untergrenze] € netto

Erstelle mir:
1. Eine Antwort, die den Preis hält (Nutzen-Argumentation)
2. Eine Antwort mit Kompromiss (Leistung reduzieren statt Preis)
3. Eine Antwort für den Fall, dass ich nachgeben muss (mit Gegenleistung)

Jede Antwort: Max. 3 Sätze. Professionell. Siezen. Kein Rechtfertigen.
```

## PROMPT 05: Testimonial-Anfrage

```
Schreibe eine E-Mail, in der ich einen zufriedenen Kunden um ein Testimonial bitte.

KUNDE: [Name, Firma]
PROJEKT: [Was wurde gemacht?]
ERGEBNIS: [Was wurde erreicht?]

Die E-Mail soll:
- Kurz sein (max. 5 Sätze)
- Das Ergebnis erwähnen (damit der Kunde weiß, worüber er schreiben soll)
- Es dem Kunden einfach machen (3 Leitfragen vorschlagen)
- Keine Pflicht suggerieren

Leitfragen für den Kunden:
1. Was war Ihre Ausgangssituation vor unserem Projekt?
2. Was hat sich durch die Zusammenarbeit konkret verändert?
3. Würden Sie mich weiterempfehlen? Wenn ja, warum?
```

## PROMPT 06: Angebotsvergleich erstellen

```
Erstelle einen internen Angebotsvergleich für mich.

Ich habe 3 potenzielle Projekte und muss priorisieren:

PROJEKT A:
- Kunde: [Name]
- Umsatz: [Betrag] €
- Aufwand: [Stunden/Tage]
- Dauer: [Wochen]
- Zahlungsbedingungen: [Vorauszahlung? Ratenzahlung?]

PROJEKT B:
- [gleiche Felder]

PROJEKT C:
- [gleiche Felder]

Bewerte nach:
1. Stundensatz (Umsatz / Aufwand)
2. Cashflow-Geschwindigkeit (wann fließt Geld?)
3. Folgeauftrag-Potenzial (1-10)
4. Aufwand-Risiko (Scope Creep Wahrscheinlichkeit)

Erstelle eine Vergleichstabelle und gib eine klare Empfehlung.
```

## PROMPT 07: Scope Creep abwehren

```
Mein Kunde hat eine Zusatzanforderung, die nicht im Angebot steht.

ORIGINAL-SCOPE: [Was wurde vereinbart?]
NEUE ANFORDERUNG: [Was will der Kunde zusätzlich?]
GESCHÄTZTER MEHRAUFWAND: [Stunden/Tage]

Schreibe eine professionelle Antwort, die:
1. Die Anforderung wertschätzt ("gute Idee")
2. Klar macht, dass es außerhalb des Scopes liegt
3. Ein Zusatzangebot macht (mit Preis und Zeitrahmen)
4. Dem Kunden die Wahl lässt

Max. 5 Sätze. Siezen. Kein Vorwurf. Kein Rechtfertigen.
```

---

## PROMPT 08: Projekt-Absage professionell formulieren

```
Ich muss ein Projekt absagen / kann ein Angebot nicht annehmen.

GRUND: [Kapazität / Budget passt nicht / Nicht mein Fachgebiet / Red Flags]
KUNDE: [Name, Firma]
PROJEKT: [Kurzbeschreibung]

Schreibe eine Absage, die:
- Höflich und wertschätzend ist
- Einen ehrlichen (aber diplomatischen) Grund nennt
- Optional einen Alternativvorschlag macht
- Die Beziehung für die Zukunft offen hält
- Max. 4 Sätze

Siezen. Professionell. Kein "leider leider".
```

## PROMPT 09: Empfehlungs-Anfrage

```
Schreibe eine E-Mail, in der ich einen bestehenden Kunden um eine Empfehlung bitte.

KUNDE: [Name, Firma]
UNSERE ZUSAMMENARBEIT: [Was wurde gemacht?]
ZIELGRUPPE FÜR EMPFEHLUNG: [Wen suche ich? z.B. "andere B2B SaaS-Unternehmen"]

Die E-Mail soll:
- Kurz sein (max. 5 Sätze)
- Auf die gute Zusammenarbeit Bezug nehmen
- Konkret sagen, wen ich suche
- Es dem Kunden einfach machen (z.B. "Falls Ihnen jemand einfällt, leiten Sie diese E-Mail gerne weiter")
- Keinen Druck aufbauen
```

## PROMPT 10: Angebots-Nachkalkulation

```
Analysiere dieses abgeschlossene Projekt:

ANGEBOTSPREIS: [Betrag] € netto
TATSÄCHLICHER AUFWAND: [Stunden]
GEPLANTER AUFWAND: [Stunden]
SCOPE CHANGES: [Ja/Nein – wenn ja, welche?]
KUNDENZUFRIEDENHEIT: [1-10]

Berechne:
1. Tatsächlicher Stundensatz
2. Abweichung Plan vs. Ist
3. Lessons Learned (was beim nächsten Mal anders machen?)
4. Empfehlung für zukünftige Preisgestaltung bei ähnlichen Projekten
```
